# MoonRail Online Ticket Booking Platform

**MoonRail** is a web-based platform for booking train tickets online. Users can browse train schedules, check seat availability, and book tickets conveniently from anywhere.

---

## Features

- **User Registration & Login**  
  Secure authentication for users to create accounts and log in.

- **Search Trains**  
  Search trains by source, destination, and date of travel.

- **Book Tickets**  
  Book tickets online and choose seat preferences.

- **Payment Integration**  
  Supports online payments through multiple methods (Credit/Debit, UPI, etc.).

- **Booking History**  
  Users can view past bookings and download tickets.

- **Admin Panel**  
  Admin can add trains, manage schedules, and monitor bookings.

---

## Technologies Used

- **Frontend:** HTML, CSS, JavaScript react js
- **Backend:** Node.js
